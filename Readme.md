## Nazar Blocker

Protect your computer from nazar.

## Prerequisites
- Qt development tools (qmake from Qt 6)
- C/C++ compiler and build tools (gcc/clang, make)
- Development packages for any library dependencies

## Build from source
1. Open a terminal in the project root.
2. Run:
```sh
qmake
make
```

## Run
- From the build directory:
```sh
./nazar
```
